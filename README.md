#jQuery.hoverOnTouch

A simple :hover emulation for touch devices


##How to use
```javascript
$('a.example').hoverOnTouch();
```

##Options
```javascript
$('a.example').hoverOnTouch({
	/*
		specify a class name that is added to element when element is touched.
		default: hover
	*/
	hoverClass: 'hover-example' 
});
```
